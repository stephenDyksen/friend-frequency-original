official_canvas.php - main code that renders the game UI in Facebook

game_functionality.js - defines most of the javascript functions for game functionality.  This file is included in official_canvas.php

ff_canvas_style.css - provides the CSS included in official_canvas.php for styling

login.php - basic code used to connect to a MySQL database using OpenShift environment tokens

mySQL_scripts.txt - commands used to build database tables

webService folder contains php files that provide the RESTful services to access the information from each of the four tables in the database

images folder contains the images used on the main site
